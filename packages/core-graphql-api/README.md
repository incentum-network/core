# ARK Core - GraphQL API

## Installation

```bash
yarn add @arkecosystem/core-graphql-api
```

## Configuration

### Defaults

```js
module.exports = {
  enabled: true,
  port: 4005,
  path: '/graphql',
  graphiql: true,
  pretty: true
}
```

## Security
